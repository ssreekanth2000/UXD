Foodwáy

Allergen Database for Restaurants

After talking to dozens of small business owners, we found a need for a digital application that organizes restaurant menu allergen information. We then designed what an app could look like for front-of-house restaurant employees to help them to answer customer questions about ingredient information.

  • Learn More

User Values

Easy to Use

The tablet app should be easily navigatable and avoid using the on-screen keyboard.

Quick to Find Information

Front-of-house employees are under a lot of pressure to find answers fast. The interface should be quick to find and easy to understand the information displayed.

Features

Menu Navigation

The categories on the left side allow the user to find the type of food they are looking for.

Allergen Filters

These toggles allow the user to quickly find items that fit the allergen criteria.

Allergen Icons

Next to each menu item is a set of icons if the food fits the allergen category or can be modified to fufil that allergen requirement.

Ingredient's Lists

This feature allows the employee to answer specific questions about what ingredients are in each menu item.

Notes Feature

The notes section allows the restaurant to make comments about each menu item pertaining to the allergens and ingredeints.

Optional Search

This feature, although tedious, still provides a useful way to quickly find specific informaiton about menu items or sort by unique allergens.
